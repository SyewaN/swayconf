## ⚡ System Info

- **OS**: Void Linux
- **WM**: Sway
- **Terminal**: Alacritty
- **Bar**: Waybar 
- **Launcher**: Fuzzel
- **Notifications**: Dunst

## ⌨️ Keybinds

| Key | Action |
|-----|--------|
| `Super + Return` | Terminal |
| `Super + D` | Fuzzel launcher |
| `Super + Q` | Close window |
| `Super + Escape` | Lock screen |
| `Super + Home` | Logout menu |
| `Super + Shift + R/T/Y` | Screenshots |

## 📁 File Structure

```
config/
```
